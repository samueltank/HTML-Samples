Tipos de layouts

Existem vários tipos de layouts, 4 para ser exato, são eles: Frameworks, float, flexbox e grid.

Para melhor explicar os tipos de layout:

Framework
É usado uma aplicação para criar layout com maior rapidez.
Float Layout
É um layout baseado em duas propriedades CSS, são elas: float e clear.
Flexbox layout
É um layout baseado no display flex, uma ótima forma de posicionar de forma dinâmica vários elementos.

Para uma estrutura semântica da página HTML podemos usar as seguintes tags:

<header>
Define um cabeçalho para um documento ou seção;
<nav>
Define um conjunto de links para navegação;
<section>
Define um seção de conteúdo em um documento;
<article>
Define um conteúdo independene do conteúdo principal;
<aside>
Define um conteúdo extra, em uma coluna a parte;
<footer>
Define o rodapé para um documento ou seção;
<details>
Define detalhes adicionais que o usuário pode abrir ou fechar sob demanda;
<summary>
Define um título para o elemento details.
